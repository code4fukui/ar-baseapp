# ar-baseapp
A base application for creating AR (Augmented Reality) experiences using Three.js and WebXR.

## Demo
The `geometry.html` file provides a simple example of using the `initAR()` function from `initAR.js` to create an AR scene with a red cube.

## Features
- Provides a boilerplate for setting up a Three.js scene with WebXR support
- Includes an `initAR()` function that sets up the necessary scene, camera, renderer, and controls
- Allows for easy integration of 3D objects and AR functionality

## Requirements
- A modern web browser with WebXR support (e.g., Chrome, Firefox, Edge)
- A device with AR capabilities (e.g., smartphone, tablet)

## Usage
To use the `ar-baseapp` project:

1. Clone the repository or download the files.
2. Open the `geometry.html` file in a compatible web browser.
3. Grant the necessary permissions to access the device's camera and AR capabilities.
4. Interact with the AR scene by moving your device around.
